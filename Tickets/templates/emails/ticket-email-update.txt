salamبه‌روزرسانی‌های سیستم تیکت
==========================

تاریخ به‌روزرسانی: 2024-01-15
نسخه: 1.0.0

تغییرات اعمال شده:

1. ایجاد قالب HTML برای نمایش تیکت در ایمیل
   - طراحی واکنش‌گرا برای نمایش در دستگاه‌های مختلف
   - استفاده از فونت‌های استاندارد و خوانا
   - رعایت جهت نوشتاری راست به چپ (RTL) برای فارسی
   - استفاده از رنگ‌بندی حرفه‌ای و مناسب

2. ایجاد قالب متنی برای ایمیل‌های متنی
   - ساختار ساده و خوانا
   - شامل تمام اطلاعات ضروری
   - فرمت‌بندی مناسب برای نمایش در کلاینت‌های مختلف ایمیل

3. ویژگی‌های قالب HTML:
   - نمایش کد پیگیری به صورت برجسته
   - بخش‌بندی مناسب اطلاعات تیکت
   - طراحی جداول با CSS بدون استفاده از تگ‌های table برای سازگاری بهتر
   - بخش متمایز برای نمایش شرح درخواست
   - فوتر مناسب با توضیحات ضروری
   - سازگاری با مرورگرهای مختلف و کلاینت‌های ایمیل

4. متغیرهای قابل استفاده در قالب:
   - {{id}}: کد پیگیری تیکت
   - {{category}}: دسته‌بندی تیکت
   - {{subject}}: موضوع تیکت
   - {{user}}: نام کاربر
   - {{description}}: شرح کامل درخواست
   - {{updated_at}}: تاریخ و زمان ایجاد تیکت

5. نکات فنی:
   - قالب HTML از CSS داخلی استفاده می‌کند (Inline CSS) برای سازگاری بیشتر با کلاینت‌های ایمیل
   - طراحی با رویکرد Mobile-first برای نمایش بهتر در موبایل
   - عدم استفاده از JavaScript برای اطمینان از نمایش در تمامی کلاینت‌ها
   - استفاده از واحدهای نسبی برای اندازه‌ها برای انعطاف‌پذیری بیشتر

6. دستورالعمل استفاده:
   - جایگزین کردن متغیرها با مقادیر واقعی در سمت سرور
   - استفاده از قالب HTML برای ایمیل‌های با قابلیت نمایش HTML
   - استفاده از قالب متنی برای ایمیل‌های متنی ساده
   - تست قالب در کلاینت‌های مختلف ایمیل قبل از استفاده عملیاتی

7. نکات ایمنی:
   - اطمینان از escape کردن محتوای کاربر قبل از جایگزینی در قالب
   - بررسی اعتبار داده‌ها قبل از قرارگیری در قالب
   - جلوگیری از تزریق کد HTML از طریق داده‌های کاربر

8. بهبودهای آینده:
   - اضافه کردن وضعیت تیکت (Status)
   - اضافه کردن اولویت تیکت (Priority)
   - اضافه کردن بخش برای پاسخ‌های پشتیبانی
   - اضافه کردن دکمه‌های عملیاتی در نسخه HTML
   - پشتیبانی از پیوست‌های تیکت
